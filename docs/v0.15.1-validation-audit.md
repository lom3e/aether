## Bug Low Priority Trovati e Decisioni Prese
1. **SemanticMemory Lifecycle (`__del__`)**
   - *Problema*: La classe `SemanticMemory` chiudeva i database `sqlite3` catturando eventuali fallback solo su `__del__`, prassi che poteva rivelarsi fragile o creare leak con GC passivo.
   - *Decisione*: Mantenere `__del__` per retrocompatibilità ma introdurre `__enter__` e `__exit__` come metodi ufficiali per garantire robustezza come *context manager*.
2. **Public API Exports**
   - *Problema*: Gli oggetti core (`Agent`, `Task`) venivano chiamati tramite un deep pathing (`aether.agents.agent.Agent`, `aether.core.execution.Task`).
   - *Decisione*: Aggiunti come export espliciti nel package root `src/aether/__init__.py` facilitando l'adozione (`from aether import Agent, Task`), senza creare dipendenze circolari in quanto i file interni importano sempre per module path assoluto.
